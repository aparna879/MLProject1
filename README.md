# IRIS DATASET
My first Machine Learning Project using Linear Regression to predict the class of the flower based on available attributes.</br>

<b>Attribute Information:</b>
1. sepal length in cm 
2. sepal width in cm 
3. petal length in cm 
4. petal width in cm 
5. class: 
-- Iris Setosa 
-- Iris Versicolour 
-- Iris Virginica
